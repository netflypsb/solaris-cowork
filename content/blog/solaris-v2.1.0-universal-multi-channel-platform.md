## 🛠️ Installation & Setup

### Quick Install (5 Minutes)
1. Download from [GitHub Releases](https://github.com/OpenCoworkAI/open-cowork/releases)
2. Run installer (Windows: `.exe`, macOS: `.dmg`)
3. Launch application and complete setup wizard
4. Configure API key and select channels
5. Start using your universal AI assistant!

### Development Setup
```bash
git clone https://github.com/OpenCoworkAI/open-cowork.git
cd open-cowork
npm install
npm run rebuild
npm run dev
```
